This plugin looks for any page that has the Blog Module enabled and applies a new style too.

== Description ==

 Divi 100 Article Card est une extension qui permet d’appliquer un nouveau style au module d’article de [Divi](https://fxbenard.com/recommande/divi/).

Ou directement par e-mail : fx@fxbenard.com.


== Installation ==

= Depuis votre site WordPress  =
Direction Extension > Ajouter > Mettre une extension en ligne > Selectionner le fichier zip préalablement téléchargé > Installer maintenant et tada !

= Installation manuelle =
Extraire le fichier zip et déposez-le dans votre dossier wp-content/plugins/ de votre installation WordPress et activez-le depuis votre page Extensions.
